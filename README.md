# Customer Segmentation & Revenue Analysis

## Overview
An end-to-end data analytics project that segments retail customers using RFM (Recency, Frequency, Monetary) analysis to identify high-value customers and provide actionable business insights.

## Live Dashboard
🔗 [View Interactive Tableau Dashboard](https://public.tableau.com/app/profile/noel.cleetus/viz/CustomerSegmentationRevenueAnalysis/CustomerSegmentationDashboard)

## Key Findings
- Analysed **500K+ transactions** from a UK-based retailer
- Identified **1,300 Champion customers** generating **$12.1M in revenue**
- Discovered **615 At-Risk customers** — previously high-value, now going quiet
- **Loyal segment (1,403 customers)** represents the largest upsell opportunity

## Tools & Technologies
- **Python** — data cleaning, feature engineering, RFM scoring
- **Pandas** — data manipulation and analysis
- **Tableau Public** — interactive dashboard and visualizations
